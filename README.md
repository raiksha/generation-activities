# Bootcamp Junior Java Developer: Repositorio de Tareas

Este repositorio contiene las tareas, ejercicios y proyectos desarrollados durante el programa intensivo de formación en desarrollo de software (Desarrollador Full Stack Jr – Java 2026 por Generation Chile). El objetivo es documentar la resolución de problemas planteados en el plan de estudios para su revisión.

---

## 📂 Estructura del Repositorio

El contenido se irá subiendo de forma cronológica por módulos semanales, en carpetas con el nombre de la semana, tal que:

* `/semana-01`
* `/semana-02`
* `/semana-03`
* `/semana-04`

En cada carpeta habrán archivos con la resolución de las actividades. Cada archivo se nombrará con la fecha en formato MM-DD más un breve título relativo a la actividad, tal que:

* `03-10_Hello_World.md`
* `03-11_MarkDown.md`
* `/03-25_CSS` -> Carpeta

---
*Este es un repositorio estrictamente académico.*
