# Actividad: Markdown

## Investigar

* Qué significa la extensión de un archivo
* Qué es Markdown
* Para qué se usan archivos .md
---
## Respuestas:

* La extensión de un archivo es un sufijo que se añade al final del nombre de un archivo, generalmente de 2 a 4 carácteres de largo, en el formato punto + sufijo (ejemplo: .docx, .md, .png), el cual sirve para identificar qué tipo de archivo es y qué aplicación debe utilizar el sistema operativo para abrirlo.
* Markdown es un lenguaje de marcado ligero que permite dar formato a textos (negrita, cursiva, títulos, etc) utilizando carácteres simples (como *, #, <>, ``, _, etc) en archivos de texto plano. Fue creado para ser rápido de escribir y de leer, y puede ser transformado a HTML con facilidad. Los archivos escritos en Markdown utilizan la extensión .md.
* Principalmente para crear documentación técnica, archivos en repositorios de código como GitHub o para contenidos en blogs o sitios web.
