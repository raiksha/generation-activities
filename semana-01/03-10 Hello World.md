# Actividad: Hello World!

## 1. Cómo escribir un "Hello World" o "Hola mundo", en tres lenguajes: JavaScript, Python y Java.

A continuación se presenta la escritura del programa "Hello World" en tres lenguajes distintos:

### **Python**
```python
print("Hello World")

```

### **JavaScript**

```javascript
console.log("Hello World");

```

### **Java**

```java
public class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Hello World");
    }
}

```

---

## 2. Preguntas

### **¿Qué hace el programa Hello World?**

**R:** Imprime en consola la frase “Hello World”.

### **¿Qué diferencias visuales notan entre JavaScript, Python y Java?**

**R:** La cantidad de carácteres y líneas de textos necesarias para obtener el mismo resultado en los distintos lenguajes.

### **¿Qué creen que significa ‘sintaxis’ después de ver estos ejemplos?**

**R:** La sintaxis son las reglas de escritura propias de cada lenguaje de programación para hacer que un código se ejecute de manera correcta.
